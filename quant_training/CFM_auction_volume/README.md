# CFM Auction Volume

Project workspace for auction volume prediction experiments inspired by quantitative trading workflows.

## Source

The project description and data can be found on the Challenge Data page:

https://challengedata.ens.fr/challenges/60

## Structure

- `notebooks/`: exploratory notebooks and final analysis
- `data/raw/`: original input CSV files
- `data/processed/`: cleaned or generated datasets
- `src/`: reusable Python modules
- `reports/`: figures, tables, and written outputs
