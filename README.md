# Quant Training

Public portfolio repository for quantitative finance training projects.

## Projects

- `LongStaff_Schwartz_Berm_Options`: Bermudan option pricing with Longstaff-Schwartz Monte Carlo, polynomial continuation regression, neural-network extension, and mathematical method notes.
- `QRT_dir_forecasting`: Directional forecasting challenge for systematic asset allocation returns, including challenge notes, benchmark workflow, engineered features, validation checks, LightGBM modeling, and submission generation.
- `CFM_Auction_Pred`: CFM auction volume prediction notebooks, including the challenge description and a full prediction pipeline with exploratory checks, feature engineering, cross-validation diagnostics, and model outputs.
- `leetcode.ipynb`: Python practice notes for algorithmic problem solving.

## Repository Notes

The public repository keeps notebooks, source code, and lightweight project notes. Large raw challenge datasets and generated outputs are intentionally excluded when possible.
